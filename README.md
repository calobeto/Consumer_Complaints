# Consumer_Complaints
A categorization of customer complaints using NLP

Task
Use a Text Classifier to predict the category of a customer complaint

Summary
The project consist on of these main parts
- Import the dataset
- Clean the dataframe : nulls, convert string dates, etc
- Do some exploratory data analysis


Note : The dataset is huge (almost 850 mb) so it must be download and then save into a data/ folder.
