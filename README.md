# Consumer_Complaints
A categorization of customer complaints using NLP

## Task
Use a Text Classifier to predict the category of a customer complaint

## Summary
The project consist on of these parts
- Import the dataset
- Clean the dataframe : nulls, convert string dates to dates, correct some product name, etc
- Do some exploratory data analysis
- Apply Word Cleaning, Feature engineering and TF-IDF vector representation
- Use Linear Support Vector machine Classifier
- Evaluate the model


Note : The dataset is huge (almost 850 mb) so it must be download and then save into a data/ folder.
